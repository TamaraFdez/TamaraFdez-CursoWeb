Formato Nombre completo Extensión Tipo Ventajas Desventajas Ejemplo
JPG Joint Photographic Experts Group .jpg .jpeg Rasterizado
  • Compatible con la mayoria de navegadores y programas
  • Puede mostrar millones de colores
  • Ideal para imagenes de gran tamaño y fotografías
  • Pierde calidad al cambiar el tamaño (se pixela)
  • No admite transparencia
PNG .png Portable Network Graphics Rasterizado
  • Admite transparencias y degradados
  • Muy pesado
GIF .gif Graphics Interchange Format Rasterizado
  • Compresión de datos con el algoritmo Lempel-Ziv-Welch
  • Tamaño ligero
  • Acepta animación
  • Baja calidad
WebP .webp WebP Rasterizado
  • Ligero
  • Permite mostrar la misma calidad que JPEG sin pérdidas de gráficos en 24 bits
  • Muchos programas estándar de retoque fotográfico no admiten WebP
  • Sólo los navegadores web modernos soportan WebP, las cámaras de fotos y teléfonos guardan las imágenes en JPEG
SVG .svg Scalable Vector Graphics Vectorial
  • Escalable, no pierde calidad aumentando o disminuyendolo
  • Tamaño minimo y bien optimizado
  • Formas limitadas
  • No es facil trabajar con ellos para editarlos
  • No son tan faciles de incrustar como .png y algunas bibliotecas multimedia no los admiten por defecto
