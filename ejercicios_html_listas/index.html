Lista de definiciones

HTTP:
Protocolo de la capa de aplicación para la transferencia de archivos de hipertexto, como html.
HTML:
HyperText Markup Languaje, es el lenguaje con el que se define el contenido de las páginas web, En resumen, se trata de un conjunto de etiquetas que sirven para definir un texto y otros elementos que componen una página web.
URL:
Dirección web
TCP/IP:
Modelo funcional del protocolo de enlade de datos que se usa en internet
W3C:
Varias organizaciones unidas en el World wide wed consortium para trabajar en desarrolar estandares de la web para mantener la accesibilidad, internacionalización privacidad y seguridad
Mozilla Foundation:
Organización sin animo de lucro creada en 1998 con la liberación del código fuente del navegador Netscape. Es una comunidad goblal que se dedica a proteger la Web como un recurso público y ayudar a sus usuarios, crea productos de código abierto, enseña habilidades del siglo 21 y estimula campañas de promoción a escala comunitaria
